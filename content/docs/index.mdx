## Introduction

Welcome to **Krave Docs** 

This documentation will guide you through using **Gemini API** and other popular **AI models** (such as OpenAI and Claude) inside a **Next.js** application. Whether you’re building a chatbot, AI-powered features, or experimenting with generative AI, Krave helps you integrate these APIs in a clear and practical way.

You’ll learn how to:
- Connect AI APIs securely in Next.js  
- Handle API requests and responses  

Each guide focuses on **simple explanations**, **real code examples**, and **best practices** so you can move from setup to production with confidence.
